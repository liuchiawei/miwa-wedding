## 優先度說明

| P   | 意義                     |
| --- | ------------------------ |
| 1   | 必做，阻塞其他功能       |
| 2   | 核心功能，婚禮前必須完成 |
| 3   | 重要，建議完成           |
| 4   | 加分項，時間允許再做     |
| 5   | 可選，婚後或第二階段     |

## 複雜度說明

| C   | 意義                  |
| --- | --------------------- |
| 1   | 半天內可完成          |
| 2   | 約 1 天               |
| 3   | 約 2–3 天             |
| 4   | 約 1 週               |
| 5   | 超過 1 週或需額外研究 |
